配置环境

  1. centos7.9
  2. 服务器配置ffmpeg
  3. 安装screen： yum -y install screen

创建进程

  1. 新建窗口：screen -S Garrick
  2. 运行bash: bash bash.sh
  3. 挂起进程
    1. 查看当前screen进程列表：screen -ls
    2. 更改状态为detached: screen -d 进程名称
    3. 关闭某个进程： screen -S 进程名称 -X quit
